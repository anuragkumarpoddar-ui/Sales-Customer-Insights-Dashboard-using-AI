# Sales-Customer-Insights-Dashboard-using-AI
The AR Solutions – Sales &amp; Customer Insights Dashboard is a data-driven analytics project developed to transform raw transactional data into meaningful business insights. This project was designed and implemented using the assistance of AI tools ChatGPT,enabling rapid development of a modern, interactive, and visually compelling dashboard similar to industry tools like Power BI.
The primary objective of this project is to support data-driven decision-making in sales, marketing, and customer experience by providing clear, interactive insights.

Key insights derived from the dashboard include:

Strong dominance of metro cities like Delhi in sales performance
Highly competitive brand distribution with no single monopoly
Rapid adoption of digital payments, especially UPI
Above-average customer satisfaction with opportunities for improvement
Actionable trends in customer behavior and purchase patterns

This project highlights my ability to combine data analytics, visualization techniques, and AI-assisted development to build practical, business-oriented solutions efficiently.
